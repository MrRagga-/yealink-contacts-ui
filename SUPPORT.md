# Support

## Public support

Use GitHub Issues for:

- bug reports
- feature requests
- release regressions
- documentation problems

Repository:

- [GitHub issues](https://github.com/MrRagga-/yealink-contacts-ui/issues)

## Security issues

For suspected security issues, follow the private reporting guidance in [SECURITY.md](./SECURITY.md).

## Before opening an issue

- verify you are on the latest tagged release
- include backend and frontend versions
- include deployment mode:
  - local dev
  - Docker Compose
  - reverse-proxied deployment
- include source type involved:
  - Google
  - CardDAV
  - Nextcloud CardDAV
